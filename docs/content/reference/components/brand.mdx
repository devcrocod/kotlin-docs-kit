# Brand

Kotlin brand marks and the signature gradient. Each specimen below is a frozen reference from `docs/contracts/preview/` — change a token and these update automatically.

## Logo

The Kotlin K mark in its three approved variants (full-color, solid black, solid white) plus the wordmark for light and dark surfaces.

## Gradient

The signature radial gradient (and its linear and soft variants) used in heroes and accent surfaces.

<Specimen src="brand-gradient" height={360} title="Kotlin gradient" />

## Tokens

| Token                  | Purpose                                               |
| ---------------------- | ----------------------------------------------------- |
| `--kt-gradient`        | Default radial gradient. Use on heroes and hero text. |
| `--kt-gradient-linear` | 225° linear gradient for backgrounds.                 |
| `--kt-gradient-soft`   | Pastel triplet for muted surfaces.                    |
| `--kt-purple-500`      | Solid Kotlin purple.                                  |
