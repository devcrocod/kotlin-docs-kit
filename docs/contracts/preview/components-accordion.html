How do I change the brand hue?

Redeclare --kt-purple-500 in a stylesheet loaded after the bundle. The body text aligns with the summary label, not the chevron.

Accordion group — bordered stack with hairline dividers, multi-open (middle item open)
Installation fails with ERR_PNPM_IGNORED_BUILDS

Approve the core-js build script or add it to pnpm-workspace.yaml under allowBuilds.

Fonts don't load under a sub-path baseURL

Absolute /fonts/ URLs 404 on sub-path deployments — the kit rewrites font URLs to ../fonts/ at build time; keep that rewrite if you fork the pipeline.

Search returns no results

The Pagefind index is built after hugo — run npx pagefind --site public or use the Taskfile target.
