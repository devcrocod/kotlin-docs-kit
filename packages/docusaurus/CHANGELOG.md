## [0.2.0](https://github.com/devcrocod/kotlin-docs-kit/compare/v0.1.0...v0.2.0) (2026-07-07)


### ⚠ BREAKING CHANGES

* Mintlify-inspired shell redesign. Breadcrumbs (.kt-crumbs) are removed in favour of a teal .kt-eyebrow; sidebar active state is teal text only; single titled code blocks emit .kt-codeblock__title instead of a lone active .kt-codeblock__tab; icons unified at stroke 1.5. See each package's "Migrating to 0.2.0" README section for the full list.
