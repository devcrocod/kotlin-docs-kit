# Design tokens

The starter inherits every CSS custom property from the kit's design tokens,
bundled inside the preset. Override any variable in your own stylesheet to re-skin
the site without touching React.

Neutral scale: `--gray-{0..950}` in 50-step increments.

`100/300/500/600` stops.

## Overriding

Pass a stylesheet via the preset's `theme.customCss` option in
`docusaurus.config.ts`:

```ts
{
  theme: {
    customCss: './src/css/brand.css',
  },
}
```

```css
:root {
  --color-primary: #ff6b35;
  --color-link: var(--color-primary);
}
```

Restart `pnpm start` — overrides apply across both palettes via the
`[data-theme]` mechanism the kit uses for dark mode.
